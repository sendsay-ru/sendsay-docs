# Прогрев IP-адресов

**Прогрев IP-адреса** — это постепенное увеличение количества отправленных email-рассылок с&nbsp;нового IP-адреса. Задача этого процесса&nbsp;&mdash; познакомить почтовые провайдеры с&nbsp;привычными объёмами отправлений. Так формируется позитивная репутация отправителя, которая влияет на&nbsp;доставляемость писем.

Прогрев IP-адреса требуется, когда&nbsp;вы:

- начинаете отправку писем с&nbsp;нового IP-адреса,
- переходите на&nbsp;другой сервис рассылок с&nbsp;выделенным&nbsp;IP,
- возобновляете отправку email-рассылок после перерыва.

:::tip Важно
Существуют понятия прогрева как IP-адреса, так и&nbsp;домена&nbsp;&mdash; технически оба процесса одинаковы. Прогрев домена необходим, когда&nbsp;вы:

- начинаете отправку писем с&nbsp;нового домена,
- имеете старый домен, но&nbsp;вы&nbsp;ещё не&nbsp;отправляли с&nbsp;него рассылок,
- переходите на&nbsp;другой сервис рассылок&nbsp;&mdash; нужно прогреть новую связку домен + IP.
  :::

Если начать отправлять большое количество писем с&nbsp;непрогретого домена или IP-адреса, почтовые сервисы могут перестать принимать ваши рассылки: они будут считать вас ненадёжным отправителем, ведь истории отправок ещё нет. Неправильная работа с&nbsp;прогревом может привести к&nbsp;тому, что письма попадут в&nbsp;спам, а&nbsp;сам домен&nbsp;&mdash; в&nbsp;чёрный список.

## Что такое IP

**IP-адрес** (Internet Protocol&nbsp;&mdash; интернет-протокол, англ.)&nbsp;&mdash; это уникальный номер, идентифицирующий устройство в&nbsp;интернете или локальной сети. Его можно сравнить с&nbsp;телефоном: как человек звонит своему собеседнику по&nbsp;номеру, так и&nbsp;компьютер обращается к&nbsp;другому устройству по&nbsp;IP-адресу.

Есть два вида IP-адресов для отправки писем:

- выделенный IP-адрес присваивается исключительно вашему аккаунту,
- общий IP-адрес присваивается нескольким пользователям и&nbsp;представляет собой пул IP-адресов.

Общий IP&nbsp;используют несколько клиентов Sendsay, поэтому репутация одного зависит от&nbsp;остальных. С&nbsp;выделенным IP-адресом дела обстоят иначе: за&nbsp;его репутацию отвечает единственный отправитель.

Sendsay за&nbsp;дополнительную плату предоставляет выделенные IP-адреса для отправки email-рассылок&nbsp;&mdash; так вы&nbsp;будете уверены, что репутация адреса чистая и&nbsp;зависит только от&nbsp;ваших активностей. Если вам необходим выделенный IP-адрес, напишите в&nbsp;техподдержку.

## Пошаговая инструкция: как прогреть IP-адрес

Перед отправкой писем для прогрева нужно выполнить технические настройки, актуализировать базу и&nbsp;подготовить письма для прогрева.

### 1. Настройте email-аутентификацию

Пропишите [ключ DKIM, SPF-записи и&nbsp;протокол DMARC](./email-authentication.md). Вы&nbsp;можете сделать это как самостоятельно, так и&nbsp;заказать полную настройку у&nbsp;специалистов Sendsay.

[Как настроить email-аутентификацию](./how-to-connect-domain/#способы-настройки-email-аутентификации)

### 2. Активируйте Feedback Loop

FBL (Feedback Loop&nbsp;&mdash; петля обратной связи, англ.)&nbsp;&mdash; это способ получения обратной связи, который помогает отслеживать жалобы на&nbsp;спам.

:::tip Важно
Жалоба на&nbsp;спам&nbsp;&mdash; это негативный сигнал для почтового провайдера. Увеличение количества жалоб ведёт к&nbsp;снижению репутации отправителя, поэтому при прогреве IP-адреса необходимо следить за&nbsp;этим показателем.
:::

FBL позволяет проанализировать содержание рассылки и&nbsp;скорректировать его, чтобы сократить количество жалоб и&nbsp;избежать блокировок. Чтобы его активировать, нужно добавить домен в&nbsp;постмастеры. Далее следуйте инструкциям:

- [в&nbsp;постмастере Mail.ru](https://help.mail.ru/developers/notes/fbl),
- [в&nbsp;Postmaster Tools by&nbsp;Gmail](https://support.google.com/mail/answer/6254652?hl=ru).

### 3. Актуализируйте базу контактов

Ключевые характеристики успешного прогрева&nbsp;&mdash; это вовлечённость и&nbsp;последовательность.
Важно наращивать количество отправленных рассылок постепенно и&nbsp;не&nbsp;отступать от&nbsp;плана. Поэтому базу контактов нужно разделить на&nbsp;группы и&nbsp;отправлять рассылки &laquo;пачками&raquo;.

Кроме того, база для прогрева должна состоять из&nbsp;вовлечённых подписчиков&nbsp;&mdash; тех, кто активно открывает письма и&nbsp;с&nbsp;наименьшей вероятностью отправит рассылку в&nbsp;спам. Поэтому перед прогревом нужно актуализировать список контактов: отсеять пассивных подписчиков, а&nbsp;активных разделить на&nbsp;группы.

Если у&nbsp;вас ещё нет базы и&nbsp;ваш email-маркетинг только на&nbsp;старте, органический прирост подписчиков создаст идеальные условия для IP-прогрева. Новые контакты в&nbsp;базе будут позитивно влиять на&nbsp;репутацию отправителя без какого-либо вмешательства.

:::tip Важно
Есть ещё один этап при актуализации базы для прогрева&nbsp;&mdash; это [валидация](https://sendsay.ru/blog/kak-provierit-pochtu-na-sushchiestvovaniie/) адресов. В&nbsp;Sendsay есть встроенная функция валидации при импорте подписчиков.
:::

[Как импортировать подписчиков](/docs/subscribers/import-and-export/how-to-import-subscribers.md)

Сегментируйте базу и&nbsp;исключите пассивных пользователей, которые не&nbsp;открывали ваши рассылки на&nbsp;протяжении трёх&mdash;четырёх месяцев. Затем разделите базу на&nbsp;группы по&nbsp;активности, чтобы прогревать её&nbsp;небольшими частями&nbsp;&mdash; от&nbsp;самых активных к&nbsp;менее активным. Это могут быть, например:

- контакты, которые открывали рассылки в&nbsp;течение последнего месяца, двух месяцев, трёх месяцев и&nbsp;так далее;
- контакты, которые переходили по&nbsp;ссылкам из&nbsp;писем в&nbsp;течение последнего месяца, двух месяцев, трёх месяцев и&nbsp;так далее.

[Что такое сегмент контактов](/docs/subscribers/lists-and-segments/what-is-segment.md)

В&nbsp;результате база может сильно сократиться, зато в&nbsp;ней будут только активные контакты и&nbsp;лояльные подписчики, заинтересованные в&nbsp;вашей рассылке. Всё это повысит уровень открытий и&nbsp;снизит количество жалоб, что важно для прогрева и&nbsp;положительной репутации отправителя у&nbsp;почтовых систем.

### 4. Подготовьте нейтральное письмо

При прогреве IP-адреса лучше не&nbsp;использовать рекламную проморассылку. Остановитесь на&nbsp;полезном контенте, который вызывает хороший отклик у&nbsp;ваших читателей. Важно:

- наличие рабочей ссылки отписки&nbsp;&mdash; иначе письмо может попасть в&nbsp;спам,
- рекомендуемый вес HTML-кода письма&nbsp;&mdash; до&nbsp;102&nbsp;Кб, чтобы оно не&nbsp;обрезалось в&nbsp;Gmail,
- в&nbsp;тексте не&nbsp;должно быть [спам-слов](https://sendsay.ru/blog/stop-slova-kotoryie-moghut-otpravit-vashu-email-rassylku-v-spam/),
- в&nbsp;рассылке должна быть корректная и&nbsp;чистая вёрстка, которая хорошо отображается на&nbsp;разных устройствах.

[Как создать письмо в&nbsp;блочном редакторе](/docs/email-campaigns/create-your-campaign/drag-and-drop-editor.md)

[Как загрузить HTML-шаблон](/docs/email-campaigns/create-your-campaign/how-to-upload-html-template.md)

Когда вы&nbsp;уверены в&nbsp;базе и&nbsp;сверстали красивое письмо, можно приступать к&nbsp;прогреву.

### 5. Составьте график и&nbsp;начните отправлять письма

Теперь нужно определиться с&nbsp;количеством отправляемых рассылок. Главное&nbsp;&mdash; не&nbsp;торопиться: отправляйте небольшие объёмы писем самым активным подписчикам, постепенно увеличивая количество получателей.

:::tip Важно
Универсального графика увеличения объёмов рассылок нет: оптимальная скорость зависит от&nbsp;того, насколько давно вы&nbsp;собираете подписную базу, активности получателей и&nbsp;ряда других факторов.
:::

Чтобы с&nbsp;большей вероятностью понравиться почтовикам, можно наращивать количество отправляемых писем на&nbsp;10% от&nbsp;предыдущего дня. Этот способ прогрева долгий, но&nbsp;самый безопасный.

Например, вы&nbsp;начинаете с&nbsp;1&nbsp;000 писем в&nbsp;день. Тогда формула для второго дня отправки будет выглядеть так:

**1&nbsp;000 + (1&nbsp;000 \* 0,1) = 1100**

А&nbsp;для третьего дня&nbsp;&mdash; так:

**1&nbsp;100 + (1&nbsp;100 \* 0,1) = 1&nbsp;210**

И&nbsp;так далее.

При этом способе прогрева используйте автоматическую отправку выпуска по&nbsp;частям:

- в&nbsp;первый день отправьте 1&nbsp;000&nbsp;писем, ограничив аудиторию при настройке выпуска. Для этого измените настройки на шаге **Аудитория**:

- начиная с&nbsp;третьей недели наращивайте объём на&nbsp;20% ежедневно.

:::tip Важно
При отправке рассылок для прогрева не&nbsp;допускайте резких скачков и&nbsp;плавно наращивайте объёмы.

### 6. Отслеживайте показатели

Регулярно отслеживайте статистику, чтобы определить оптимальную частоту рассылки. Какие показатели здесь важны:

- ошибки доставки (Ошибки&nbsp;%) должны быть в&nbsp;пределах 1&ndash;1,5%;
- соотношение отписок к&nbsp;открытиям (UTOR&nbsp;%) должно быть не&nbsp;больше 4%.

В&nbsp;Sendsay данные по&nbsp;этим метрикам можно посмотреть в&nbsp;разделе **Статистика**. Выберите нужный выпуск и&nbsp;в&nbsp;открывшейся статистике перейдите во&nbsp;вкладку **Доставка &rarr; Ошибки доставки**. Во&nbsp;вкладке **Отписки** можно посмотреть количество и&nbsp;процент пользователей, отписавшихся от&nbsp;рассылки через жалобу на&nbsp;спам.

Если вы&nbsp;смогли достичь высокой доставляемости, попробуйте ускорить процесс прогрева. Если у&nbsp;писем с&nbsp;нового IP-адреса мало открытий и&nbsp;много отписок, лучше снизить темп.

Также следите за&nbsp;статистикой постмастеров: идеально, если все письма имеют статус &laquo;Доставлено&raquo;, а&nbsp;репутация остается в&nbsp;зелёной зоне. Если&nbsp;же доставленных писем вообще нет, значит почтовик внёс вас в&nbsp;чёрный список&nbsp;&mdash; тогда следует обратиться в&nbsp;техподдержку постмастера и&nbsp;следовать рекомендациям.
