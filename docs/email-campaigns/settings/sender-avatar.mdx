# Как настроить аватар отправителя

Аватар — это картинка, которая отображается рядом с именем отправителя и темой письма (как правило, это фотография человека или логотип компании). Если аватар не настроен, почтовый сервис подставит вместо него одну или две буквы, сгенерированные из имени отправителя.

![How avatars look](/img/email-campaigns/settings\sender-avatar/how-avatars-look.png) <br/>

## Способы настройки аватара

Их всего два:

1. Изменить аватар на стороне почтового сервиса (для каждого сервиса его нужно настраивать отдельно).
2. Добавить в настройки DNS-сервера запись BIMI, которая будет прикреплять логотип к вашим письмам. Для этого на домене должны быть настроены DKIM, SPF и DMARC.

[Как настроить служебные заголовки](https://docs.sendsay.ru/email-campaigns/settings/how-to-connect-domain)

Второй способ пока поддерживают не все почтовые сервисы (только Gmail, Yahoo, AOL и Verizon), поэтому для Яндекса и Mail.ru аватары придётся настраивать отдельно. Рассмотрим настройку аватаров для Mail.Ru, Яндекса, Gmail и отдельно расскажем, как добавить запись BIMI.

:::tip Важно
После загрузки аватар проходит модерацию — это может занять до нескольких дней.
:::

## Как добавить параметр BIMI

BIMI — это запись в настройках DNS, которая помогает почтовым сервисам идентифицировать отправителя и добавляет в рассылку логотип бренда. Для настройки BIMI DMARС-политика должна быть в блокирующем режиме (p=quarantine или p=reject).

Также перед добавлением BIMI рекомендуем получить сертификат VMC — это цифровой сертификат, который подтверждает право собственности на логотип. Без него некоторые почтовые сервисы (например, Gmail) не будут показывать ваш логотип.

[Как получить сертификат VMC](https://support.google.com/a/answer/10911028?hl=ru)

BIMI добавляется в три шага:

### 1. Добавьте логотип на сайт

Загрузите на общедоступный веб-сервер вашего домена квадратный логотип формата SVG 1.2 Tiny Portable/Secure. Требования к логотипу:

- соотношение сторон — 1:1,
- изображение должно располагаться по центру,
- фон должен быть непрозрачным.
- максимальный размер SVG-файла — 32 кб.

:::tip Важно
Рекомендуем оставлять место вокруг изображения, так как почтовые сервисы по-разному показывают логотипы и могут их обрезать.
:::

![BIMI logo](/img/email-campaigns/settings\sender-avatar/bimi-logo.png) <br/>

В базе знаний Gmail есть подробная статья, как подготовить логотип в указанном формате для настройки BIMI:

[Как создать и загрузить файл с логотипом бренда](https://support.google.com/a/answer/10911027?hl=ru-UA&ref_topic=10911234)

### 2. Сгенерируйте запись BIMI

1. Откройте генератор BIMI и нажмите «Generate BIMI».

2. Вставьте в форму:

   - ссылку на ваш домен,
   - ссылку на файл SVG,
   - ссылку на сертификат VMC (если есть).

3. Нажмите «Generate BIMI record» — создание записи может занять несколько минут. Когда запись будет готова, скопируйте её.

### 3. Добавьте BIMI в настройки DNS

1. Перейдите на сайт провайдера, у которого находится DNS-зона управления вашим доменом, и войдите в «Панель управления».

2. Откройте раздел управления DNS-зонами домена, добавьте TXT-запись и вставьте в неё значения из сгенерированной записи BIMI:

   - в поле host — `default._bimi.вашдомен`,
   - в поле value — остальную часть записи,
   - в поле TTL можно оставить значение по умолчанию (3600) — это интервал в секундах, после которого изменения в записях DNS-сервера вступят в силу. Не рекомендуем без надобности задавать меньшие значения — это увеличит нагрузку на сервер и замедлит обработку данных.

3. Обновите запись DNS — это может занять до 48 часов.

### 4. Проверьте, что всё настроено правильно

На странице [генератора BIMI](https://bimigroup.org/bimi-generator/) можно протестировать, правильно ли вы добавили параметр в настройки DNS. Для этого введите адрес домена и нажмите «Check BIMI» — сайт сгенерирует отчёт, который покажет, корректно ли указаны в настройках DNS все необходимые записи для BIMI и отображается ли сам логотип.
