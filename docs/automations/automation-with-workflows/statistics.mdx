# Статистика сценариев

Без аналитики невозможно построить эффективный email-маркетинг: анализ результатов поможет понять, решают&nbsp;ли email-кампании поставленные задачи.

[Всё про статистические отчёты](https://docs.sendsay.ru/statistics/all-about-campaign-reports/)

Статистика на&nbsp;странице сценария расскажет об&nbsp;отправленных выпусках и&nbsp;подписчиках, которые проходят или уже завершили сценарий. Чтобы её&nbsp;посмотреть, перейдите в&nbsp;боковом меню в&nbsp;раздел **Автоматизации&nbsp;&rarr; Сценарии** и&nbsp;выберите нужный сценарий. Статистические данные доступны во&nbsp;вкладках **Сводка** и&nbsp;**Просмотр участников сценария**.

## Статистика по&nbsp;сценарию

Во&nbsp;вкладке **Сводка** на&nbsp;странице сценария выводится информация о&nbsp;его участниках:

![Workflow statistics](/img/automations/automation-with-workflows/statistics/workflow-statistics.png)

- **Проходят сценарий**&nbsp;&mdash; количество подписчиков, которые попали в&nbsp;сценарий и&nbsp;стоят на&nbsp;одном из&nbsp;блоков цепочки;
- **Завершений сценария**&nbsp;&mdash; количество подписчиков, которые прошли все блоки сценария или для кого сценарий завершился принудительно.

## Статистика выпусков сценария

Ниже во&nbsp;вкладке **Сводка** выводится статистика по&nbsp;отправленным в&nbsp;сценарии выпускам. Здесь&nbsp;же можно отфильтровать таблицу по&nbsp;периоду и&nbsp;экспортировать результаты в&nbsp;формате xlsx или csv&nbsp;&mdash; для этого нажмите &laquo;Экспорт&raquo;.

![Email statistics](/img/automations/automation-with-workflows/statistics/email-statistics.png)

Статистика по&nbsp;выпускам сценария также доступна в&nbsp;разделе **Статистика &rarr; Триггерные рассылки**.

[Где следить за статистикой](https://docs.sendsay.ru/statistics/where-to-track-statistics)

## Данные об&nbsp;участниках сценария

Во&nbsp;вкладке **Просмотр участников сценария** на&nbsp;странице сценария выводятся статусы всех его участников:

![View subscribers](/img/automations/automation-with-workflows/statistics/member-preview.png)

- **Проходит сценарий**&nbsp;&mdash; участник попал в&nbsp;сценарий и&nbsp;находится на&nbsp;одном из&nbsp;его шагов;
- **Исключен принудительно**&nbsp;&mdash; сценарий завершился для участника принудительно. Здесь&nbsp;же указывается причина&nbsp;&mdash; каким образом сценарий был завершен принудительно;
- **Завершено**&nbsp;&mdash; участник прошел сценарий до&nbsp;конца.

Здесь же красным маркером отмечены подписчики, недоступные для рассылки — причину недоступности можно посмотреть, наведя курсором на адрес участника сценария.

В&nbsp;строке поиска можно найти нужный контакт по&nbsp;его email-адресу. **Дата последнего действия** напротив каждого контакта означает время попадания на&nbsp;текущий шаг или время завершения сценария. Эти данные можно отфильтровать по&nbsp;дате.

Вы&nbsp;также можете отфильтровать таблицу по&nbsp;статусу прохождения сценария&nbsp;&mdash; актуально, если необходимо посмотреть подписчиков, которые завершили сценарий или находятся на&nbsp;одном из&nbsp;его шагов. Для этого нажмите «Все подписчики» и&nbsp;выберите нужный пункт:

![Subscriber filter](/img/automations/automation-with-workflows/statistics/subscriber-filter.png)

### Какие действия можно выполнить с&nbsp;участниками сценария

Чтобы выполнить какое-либо действие с&nbsp;участниками сценария, выделите нужные контакты во&nbsp;вкладке **Просмотр участников сценария** или отметьте сразу всех:

![How to select subscribers](/img/automations/automation-with-workflows/statistics/how-to-select-subscribers.png)

Здесь можно:

- Принудительно завершить сценарий для выбранных подписчиков. Для этого нажмите кнопку &laquo;Завершить сценарий&raquo;;

  ![How to complete workflow](/img/automations/automation-with-workflows/statistics/how-to-complete-workflow.png)

- Добавить выбранных подписчиков в&nbsp;список. Для этого нажмите &laquo;Завершить сценарий&raquo; и&nbsp;выберите пункт **Добавить в&nbsp;список**, затем выберите нужный список;

  ![How to add subscriber to the list](/img/automations/automation-with-workflows/statistics/how-to-add-subscriber-to-the-list.png)

- Удалить выбранные контакты из&nbsp;базы. Для этого нажмите &laquo;Завершить сценарий&raquo; и&nbsp;выберите пункт **Удалить из&nbsp;базы**.

  ![How to delete subscriber](/img/automations/automation-with-workflows/statistics/how-to-delete-subscriber.png)

### Сценарии в карточке подписчика

При клике на адрес участника сценария во&nbsp;вкладке **Просмотр участников сценария** вы попадёте в [карточку подписчика](https://docs.sendsay.ru/subscribers/subscriber-data/subscriber-profile). Здесь во вкладке **Сценарии** можно увидеть текущие и пройденные подписчиком сценарии:

![Subscription card](/img/automations/automation-with-workflows/statistics/subscription-card.png)

В каждом сценарии указывается статус участника в сценарии и количество прохождений, а при нажатии на кнопку «Детали» можно увидеть историю прохождения конкретного сценария.

Если подписчик проходит сценарий в текущий момент, из карточки подписчика можно перейти на шаг в сценарии — для этого нажмите ссылку «Проходит сценарий»:

![Current workflow](/img/automations/automation-with-workflows/statistics/current-workflow.png)

Здесь же можно поставить прохождение текущего сценария на паузу или принудительно его завершить для этого подписчика. Для этого нажмите «Детали», а затем соответствующую кнопку:

![Actions with subscriber](/img/automations/automation-with-workflows/statistics/actions-with-subscriber.png)
