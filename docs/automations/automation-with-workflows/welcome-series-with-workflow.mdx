# Как настроить приветственную серию писем: пошаговая инструкция

Приветственная серия писем (или welcome-серия) — это последовательность писем, которая автоматически отправляется пользователям после [подтверждения подписки на&nbsp;рассылку](https://docs.sendsay.ru/faq/double-opt-in).

В&nbsp;первые дни после подписки пользователи наиболее активны и&nbsp;заинтересованы в&nbsp;контенте от&nbsp;бренда, поэтому компании настраивают цепочки приветственных писем, чтобы расположить к&nbsp;себе новых подписчиков, наладить коммуникацию и&nbsp;удерживать их&nbsp;внимание в&nbsp;будущем.

Чтобы настроить приветственную серию, необходимо заранее подготовить шаблоны писем и&nbsp;продумать условия отправки. Рекомендуем отправлять первое приветственное письмо новому подписчику сразу после подтверждения подписки.

[Активация через письмо-подтверждения](https://docs.sendsay.ru/subscribers/contacts/how-to-activate-inactive-contacts#активация-через-письмо-подтверждения)

Чтобы настроить приветственную серию писем:

## 1. Создайте сценарий

В&nbsp;боковом меню перейдите в&nbsp;раздел **Автоматизации &rarr; Сценарии** и&nbsp;нажмите синюю кнопку &laquo;+&nbsp;Создать сценарий&raquo;:

![How to create a workflow](/img/automations/automation-with-workflows/welcome-series-with-workflow/how-to-create-a-workflow.jpg)

После этого вы&nbsp;попадёте на&nbsp;страницу сценария. Перейдите в&nbsp;визуальный редактор, нажав кнопку &laquo;Редактировать&raquo;:

![How to edit the workflow](/img/automations/automation-with-workflows/welcome-series-with-workflow/how-to-edit-the-workflow.jpg)

## 2. Выберите стартовое событие

Сценарий приветственной цепочки писем запускают события **Подтверждение подписки** и&nbsp;**Подписчик добавлен подтверждённым**. Добавьте их&nbsp;в&nbsp;качестве Стартового события с&nbsp;помощью кнопок &laquo;Выберите событие&raquo; и &laquo;+&nbsp;Добавить&raquo;:

![How to add events](/img/automations/automation-with-workflows/welcome-series-with-workflow/how-to-add-events.gif)

## 3. Добавьте в&nbsp;сценарий первое письмо

Для этого нажмите кнопку плюса `+` под блоком Стартового события и&nbsp;выберите Действие **Отправить подписчику email**. Затем выберите шаблон первого приветственного письма&nbsp;&mdash; эта рассылка отправится новому подписчику сразу после того, как он&nbsp;подтвердит свою подписку. Нажмите кнопку &laquo;Сохранить&raquo;:

![How to add actions](/img/automations/automation-with-workflows/welcome-series-with-workflow/how-to-add-actions.gif)

## 4. Настройте время отправки второго письма

Чтобы следующее письмо приветственной цепочки пришло новому подписчику спустя некоторое время, используйте блок Таймеры. Для этого нажмите на&nbsp;кнопку плюса `+` в&nbsp;сценарии, выберите таймер **Прошло время** и&nbsp;укажите дни, часы или минуты:

![How to add timers](/img/automations/automation-with-workflows/welcome-series-with-workflow/how-to-add-timers.gif)

## 5. Добавьте в&nbsp;сценарий другие письма цепочки

Чтобы дополнить приветственную цепочку другими письмами, используйте блоки Действий и&nbsp;Таймеров. Например, если ваша цепочка состоит из&nbsp;трёх писем, последовательность добавления блоков может быть такой:

```
Стартовое событие → Приветственное письмо №1 → Прошло 2 дня → Приветственное письмо №2 → Прошло 3 дня → Приветственное письмо №3 → Конец сценария
```

## 6. Сохраните сценарий

Когда сценарий приветственной цепочки писем готов, нажмите кнопку &laquo;Сохранить&raquo; в&nbsp;правом верхнем углу визуального редактора.

![How to save the workflow](/img/automations/automation-with-workflows/welcome-series-with-workflow/how-to-save-the-workflow.png)

Затем выйдите из&nbsp;редактора, нажав стрелочку `←` в&nbsp;верхнем левом углу.

## 7. Протестируйте сценарий

Это необязательный пункт, но&nbsp;так вы&nbsp;сможете убедиться, что сценарий работает так, как вы ожидаете. Чтобы протестировать сценарий, вручную добавьте свой email в&nbsp;число участников. Для этого:

1. Перейдите на&nbsp;страницу сценария и&nbsp;нажмите зелёную кнопку &laquo;Активировать&raquo;.
2. Нажмите на&nbsp;кнопку &laquo;Действия&raquo; и&nbsp;выберите пункт **Управление участниками**.
3. Укажите свой email и&nbsp;нажмите кнопку &laquo;Добавить&raquo;:
   ![Member management](/img/automations/automation-with-workflows/welcome-series-with-workflow/member-management.gif)
   [Как протестировать сценарий](https://docs.sendsay.ru/automations/automation-with-workflows/workflow-creation/#как-протестировать-сценарий)

:::tip Важно
Новые участники, добавленные в&nbsp;сценарий вручную, попадают на&nbsp;первый шаг сценария, игнорируя Стартовое событие.
:::

## 8. Включите приём подписчиков

Когда всё готово, начните принимать в&nbsp;сценарий новых подписчиков. Для этого на&nbsp;странице сценария нажмите на&nbsp;переключатель у&nbsp;пункта **Принимать подписчиков**&nbsp;&mdash; так новые участники начнут попадать в&nbsp;сценарий.

Чтобы подписчик не&nbsp;получал одни и&nbsp;те&nbsp;же письма из&nbsp;приветственной цепочки, отключите пункт **Повторные прохождения**.

![Workflow settings](/img/automations/automation-with-workflows/welcome-series-with-workflow/workflow-settings.png)
