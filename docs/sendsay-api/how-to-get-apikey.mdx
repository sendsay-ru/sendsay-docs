# Как получить API-ключ

API-ключ — это уникальный ключ для авторизации при работе с [Sendsay API](https://sendsay.ru/api/). Чтобы его получить:

1. В&nbsp;меню аккаунта выберите **Настройки системы** и&nbsp;перейдите к&nbsp;разделу [**Пользователи**](https://app.sendsay.ru/account/sublogins).
2. API-ключ можно сгенерировать только для сублогина, поэтому необходимо добавить нового пользователя.

   Для этого нажмите &laquo;Добавить пользователя&raquo;, в&nbsp;новом окне введите имя, логин, email и&nbsp;пароль. Затем выберите роль с&nbsp;правами на&nbsp;отправку рассылок (например, **Менеджер**) и&nbsp;нажмите &laquo;Добавить пользователя&raquo;:

   ![New user](/img/sendsay-api/how-to-get-apikey/new-user.png)

   [Как создать учётную запись для нового пользователя](https://docs.sendsay.ru/account-settings/how-to-create-sublogin)

3. Выйдите из&nbsp;системы, а&nbsp;затем авторизуйтесь в&nbsp;Sendsay под только что созданным логином и&nbsp;паролем&nbsp;&mdash; чтобы подтвердить новый саблогин.
4. Снова выйдите из&nbsp;системы и&nbsp;авторизуйтесь под основным аккаунтом.
5. Перейдите в&nbsp;раздел [**Пользователи**](https://app.sendsay.ru/account/sublogins) и&nbsp;выберите созданный саблогин. Нажмите &laquo;Редактировать&raquo;, затем перейдите во&nbsp;вкладку **Настройка пользователя**.
6. В&nbsp;разделе **API-ключ** нажмите &laquo;Сгенерировать ключ&raquo;. Скопируйте его для использования в&nbsp;запросах.<br/><br/>
   ![How to get API-key](/img/sendsay-api/how-to-get-apikey/how-to-get-apikey.gif)

7. Для использования API-ключа укажите его

- внутри запроса:

<!-- prettier-ignore -->
```js
{
  "apikey": "API-ключ",
  // данные для вызова
}
```

- или в HTTP-заголовке `Authorization`:

```
Authorization: sendsay apikey=API-ключ
```

:::tip Важно
Невозможно удалить саблогин, для которого сгенерирован API-ключ: чтобы удалить саблогин, сначала удалите API-ключ.
:::
