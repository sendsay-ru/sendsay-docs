# Как получить API-ключ

API-ключ — это уникальный ключ для авторизации при работе с [Sendsay API](https://sendsay.ru/api/). В целях безопасности API-ключ нельзя сгенерировать для владельца аккаунта, поэтому нужно создать нового пользователя и сгенерировать API-ключ. Чтобы это сделать:

1. В&nbsp;меню аккаунта выберите **Настройки системы** и&nbsp;перейдите к&nbsp;разделу [**Пользователи**](https://app.sendsay.ru/account/sublogins).
2. Нажмите &laquo;Добавить пользователя&raquo;, в&nbsp;новом окне введите имя, логин, email и&nbsp;пароль. Затем выберите роль с&nbsp;правами на&nbsp;отправку рассылок (например, **Менеджер**) и&nbsp;нажмите &laquo;Добавить пользователя&raquo;:

   ![New user](/img/sendsay-api/how-to-get-apikey/new-user.png)

   [Как создать учётную запись для нового пользователя](https://docs.sendsay.ru/account-settings/how-to-create-sublogin)

3. В карточке нового пользователя перейдите во&nbsp;вкладку **Настройка пользователя**.
4. В&nbsp;разделе **API-ключ** нажмите &laquo;Сгенерировать ключ&raquo;. Скопируйте его для использования в&nbsp;запросах.<br/><br/>
   ![How to get API-key](/img/sendsay-api/how-to-get-apikey/how-to-get-apikey.gif)

5. Для использования API-ключа укажите его

- внутри запроса:

<!-- prettier-ignore -->
```js
{
  "apikey": "API-ключ",
  // данные для вызова
}
```

- или в HTTP-заголовке `Authorization`:

```
Authorization: sendsay apikey=API-ключ
```

:::tip Важно
Невозможно удалить пользователя, для которого сгенерирован API-ключ: чтобы удалить пользователя, сначала удалите API-ключ.
:::
