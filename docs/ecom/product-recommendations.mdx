# Товарные рекомендации

**Товарные рекомендации** — это блок в email-рассылке с карточками товаров, который формируется на&nbsp;основе поведения подписчика на&nbsp;сайте, его предпочтений, а&nbsp;также интересов других пользователей сайта.

Вам наверняка встречались такие блоки в&nbsp;рассылках или на&nbsp;сайтах интернет-магазинов: &laquo;Вам может понравиться&raquo;, &laquo;Специально для вас&raquo;, &laquo;С&nbsp;этим товаром покупают&raquo; и&nbsp;прочие.

<p align="center">
  <img width="50%" src={productRecommendations} alt="Product recommendations" />
</p>

Товарные рекомендации сокращают время на&nbsp;поиски нужной позиции в&nbsp;интернет-магазине и&nbsp;облегчают процесс покупки за&nbsp;счёт персонализированных предложений. Поэтому они помогают повысить интерес со&nbsp;стороны покупателей, конверсию и&nbsp;средний чек.

Чтобы добавить товарные рекомендации в&nbsp;письмо, нужно подключить модуль Продажи и&nbsp;каталог товаров. Как это сделать&nbsp;&mdash; читайте в&nbsp;статьях по&nbsp;ссылкам ниже.

[Модуль Продажи: что это и как подключить](https://docs.sendsay.ru/ecom/how-to-connect-ecom)<br/>
[Как подключить каталог товаров](https://docs.sendsay.ru/ecom/product-catalog)

:::tip Важно
Товарные рекомендации строятся на&nbsp;основе статистики: система отслеживает данные пользователей и&nbsp;их&nbsp;действия на&nbsp;сайте&nbsp;&mdash; просмотры страниц, интерес к&nbsp;категориям, добавление товаров в&nbsp;корзину, историю покупок и&nbsp;так далее. Поэтому товарные рекомендации будут правильно работать спустя время после настройки передачи данных с&nbsp;сайта в&nbsp;Sendsay&nbsp;&mdash; когда накопится статистика.
:::

В&nbsp;Sendsay есть несколько типов рекомендаций с&nbsp;готовыми шаблонами товарных блоков:

- самые продаваемые товары,
- самые продаваемые товары по&nbsp;общей сумме продаж,
- самые продаваемые товары в&nbsp;категории,
- самые продаваемые товары в&nbsp;категории по&nbsp;сумме продаж,
- самые просматриваемые товары,
- самые просматриваемые товары в&nbsp;категории.

## Как добавить товарные рекомендации в&nbsp;письмо

:::tip Важно
Товарные рекомендации можно добавить в&nbsp;любую рассылку. Чтобы в&nbsp;письма подгружались товары из&nbsp;каталога, необходимо, чтобы id&nbsp;товара, который передаётся в&nbsp;Sendsay в&nbsp;событии, совпадал с&nbsp;id&nbsp;товара в&nbsp;файле каталога товаров.
:::

Чтобы товарные рекомендации заработали, в&nbsp;письмо нужно добавить специальный блок&nbsp;&mdash; шаблоны вы&nbsp;найдете в&nbsp;папке сайта в&nbsp;разделе **Контент**.

Для добавления товарных рекомендаций через блочный редактор выберите HTML-блок и&nbsp;задайте отступ 0&nbsp;слева и&nbsp;справа. Затем пропишите в&nbsp;нём нужную команду подстановки:

- для блока **Самые продаваемые товары**<br/>
  `[% draft("id","mysite.ru_bestseller_count") %]`
- для блока **Самые продаваемые товары** по общей сумме продаж<br/>
  `[% draft("id","mysite.ru_bestseller_money") %]`
- для блока **Самые просматриваемые товары**<br/>
  `[% draft("id","mysite.ru_eyecatcher") %]`
- для блока **Самые продаваемые товары** в категории<br/>
  `[% draft("id","mysite.ru_bestseller_count_product_category") %]`
- для блока **Самые продаваемые товары** в категории по общей сумме продаж<br/>
  `[% draft("id","mysite.ru_bestseller_money_product_category") %]`
- для блока **Самые просматриваемые товары** в категории<br/>
  `[% draft("id","mysite.ru_eyecatcher_category_view") %]`

где `mysite.ru` — название вашего сайта. Например, для&nbsp;сайта Sendsay код блока самых продаваемых товаров будет выглядеть&nbsp;так:

```
[% draft("id","sendsay.ru_bestseller_count") %]
```
