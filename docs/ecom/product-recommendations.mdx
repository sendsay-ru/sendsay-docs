# Товарные рекомендации

**Товарные рекомендации** — это блок в email-рассылке с карточками товаров, который формируется на основе поведения подписчика на сайте, его предпочтений, а также интересов других пользователей сайта.

Вам наверняка встречались такие блоки в рассылках или на сайтах интернет-магазинов: «Вам может понравиться», «Специально для вас», «С этим товаром покупают» и прочие.

Товарные рекомендации сокращают время на поиски нужной позиции в интернет-магазине и облегчают процесс покупки за счёт персонализированных предложений. Поэтому они помогают повысить интерес со стороны покупателей, конверсию и средний чек.

Чтобы добавить товарные рекомендации в письмо, нужно подключить модуль «Sendsay Продажи» и каталог товаров. Как это сделать — читайте в статьях по ссылкам ниже.

[Модуль «Sendsay Продажи»: что это и как подключить](https://docs.sendsay.ru/ecom/how-to-connect-ecom)<br /> [Как подключить каталог товаров](https://docs.sendsay.ru/ecom/product-catalog)

:::tip Важно

Товарные рекомендации строятся на основе статистики: система отслеживает данные пользователей и их действия на сайте — просмотры страниц, интерес к категориям, добавление товаров в корзину, историю покупок и так далее. Поэтому товарные рекомендации будут правильно работать спустя время после настройки передачи данных с сайта в Sendsay — когда накопится статистика.

:::

В Sendsay есть несколько типов рекомендаций с готовыми шаблонами товарных блоков:

- самые продаваемые товары,
- самые продаваемые товары по общей сумме продаж,
- самые продаваемые товары в категории,
- самые продаваемые товары в категории по сумме продаж,
- самые просматриваемые товары,
- самые просматриваемые товары в категории.

## Как добавить товарные рекомендации в письмо

:::tip Важно

Товарные рекомендации можно добавить в любую рассылку. Чтобы в письма подгружались товары из каталога, необходимо, чтобы id товара, который передаётся в Sendsay в событии, совпадал с id товара в файле каталога товаров.

:::

Чтобы товарные рекомендации заработали, в письмо нужно добавить специальный блок — шаблоны вы найдете в папке сайта в разделе **Контент**.

Для добавления товарных рекомендаций через блочный редактор выберите HTML-блок и задайте отступ 0 слева и справа. Затем пропишите в нём нужную команду подстановки:

- для блока **Самые продаваемые товары**<br /> `[% draft("id","mysite.ru_bestseller_count") %]`
- для блока **Самые продаваемые товары** по общей сумме продаж<br /> `[% draft("id","mysite.ru_bestseller_money") %]`
- для блока **Самые просматриваемые товары**<br /> `[% draft("id","mysite.ru_eyecatcher") %]`
- для блока **Самые продаваемые товары** в категории<br /> `[% draft("id","mysite.ru_bestseller_count_product_category") %]`
- для блока **Самые продаваемые товары** в категории по общей сумме продаж<br /> `[% draft("id","mysite.ru_bestseller_money_product_category") %]`
- для блока **Самые просматриваемые товары** в категории<br /> `[% draft("id","mysite.ru_eyecatcher_category_view") %]`

где `mysite.ru` — название вашего сайта. Например, для сайта Sendsay код блока самых продаваемых товаров будет выглядеть так:

```
[% draft("id","sendsay.ru_bestseller_count") %]
```
