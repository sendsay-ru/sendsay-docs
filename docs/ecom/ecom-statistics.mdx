# Статистика в модуле Продажи

Статистика помогает оценить эффективность рассылок, выявить ошибки и усовершенствовать стратегию. В базе знаний есть отдельная [статья про статистические отчёты](https://docs.sendsay.ru/statistics/all-about-campaign-reports) — эти показатели помогут вносить изменения и улучшать рассылки.

## Где смотреть статистику по сайту и выпускам

Статистика по сайту и выпускам триггерных сценариев выводятся во вкладке **Сводка** на странице сайта:

![Summary](/img/ecom/ecom-statistics/summary.png)

В статистике по сайту выводятся данные:

- Уникальные контакты — это число подписчиков, которые посещают сайт и чьи данные мы знаем. Например, когда подписчик переходит на сайт из письма, мы можем отследить историю событий и его поведение на сайте.

  [Как посмотреть историю событий сайта](https://docs.sendsay.ru/ecom/how-to-connect-ecom#как-посмотреть-историю-событий-сайта)

- Анонимные контакты — те пользователи, которые «гуляют» по сайту и данных о которых у нас нет. Например, они не подписаны на рассылку или могли перейти на сайт не из письма.

  Если настроена передача событий **Регистрация на сайте** и **Авторизация на сайте**, в Sendsay будут передаваться email-адреса пользователей, которые зарегистрировались или авторизовались на сайте. Система сохранит такой адрес и свяжет с ним все поступающие события этого пользователя — таким образом анонимные контакты перестанут быть анонимными и станут участниками сценария.

:::tip Важно

Система хранит данные анонимных пользователей. Когда анонимный контакт перейдёт из письма на сайт, данные соединятся, и станет доступна история событий для подписчика.

:::

- Сделано заказов — количество совершенных заказов.
- Сумма — общая сумма покупок всех совершенных заказов.

Ниже — общая сводка по рассылкам, где выводится статистика по отправке, доставке, открытиям и кликам всех выпусков сценария. Статистика по каждому отправленному выпуску выводится в таблице — её можно экспортировать в формате `.xlsx` или `.csv`:

![Mailing list summary](/img/ecom/ecom-statistics/mailing-list-summary.png)

Здесь можно настроить фильтр по периоду и увидеть статистику по выпускам за сегодня, вчера, последнюю неделю, месяц или за всё время:

![Period filter](/img/ecom/ecom-statistics/period-filter.png)

Также можно отфильтровать таблицу по выпускам — чтобы посмотреть статистику конкретного письма:

![Mailing list filter](/img/ecom/ecom-statistics/mailing-list-filter.gif)

## Статистика по заказам и выручке

Если вы используете [полную версию модуля «Продажи](https://docs.sendsay.ru/ecom/how-to-connect-ecom#возможности-версий)», в таблице по отправленным выпускам вам доступна статистика по заказам и выручке:

![Statistics on orders and revenue](/img/ecom/ecom-statistics/statistics-on-orders-revenue.png)

- Заказы — сколько заказов сделали подписчики, перешедшие на сайт из каждого конкретного письма.
- Выручка — этот показатель выводит сумму покупок в денежном эквиваленте, которые совершили подписчики, перешедшие на сайт из каждого конкретного письма.

Таким образом можно определить, сколько заказов и денег принесла каждая рассылка сценария. Статистику по заказам и выручке можно найти также во вкладке **Сводка** на странице сценария и в разделе **Статистика**.

Сводка по заказам и выручке также выводится в отчёте выпуска и в шаблоне — для пользователей бесплатной версии модуля «Продажи Cтарт» данные доступны только за последние 48 часов.

:::tip Важно

Если вы используете бесплатную версию модуля «Продажи Старт», статистика по заказам и выручке в сводной таблице рассылок будет недоступна. Чтобы в статистике выводились эти показатели, необходимо перейти на версию модуля «Продажи» за дополнительную плату — для этого напишите в чат техподдержки.

[Таблица сравнения версий](https://docs.sendsay.ru/ecom/how-to-connect-ecom/#возможности-версий)

:::

## Статистика по сценарию

Помимо статистических данных по сайту и выпускам, система выводит статистику по каждому сценарию — во вкладке **Сводка** на странице сценария:

![Workflow statistics](/img/ecom/ecom-statistics/workflow-statistics.png)

Здесь в блоке **Статистика по сценарию** выводятся данные о его участниках — сколько подписчиков проходят и завершили сценарий. А в блоке **Продажи** можно увидеть, сколько заказов и на какую сумму принёс сценарий (для пользователей бесплатной версии модуля «Продажи Cтарт» данные доступны только за последние 48 часов).

В сводной таблице по отправленным выпускам также выводятся данные о заказах и выручке по каждому конкретному выпуску (для пользователей полной версии модуля «Продажи»). Эту таблицу можно отфильтровать по периоду, по конкретному письму, отправленному в сценарии, а также экспортировать в формате `.xlsx` или `.csv`:

![Mailing list filter](/img/ecom/ecom-statistics/mailing-list-filter.png)
