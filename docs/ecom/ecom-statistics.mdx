# Статистика в модуле Продажи

Статистика помогает оценить эффективность рассылок, выявить ошибки и усовершенствовать стратегию. В базе знаний есть отдельная [статья про статистические отчёты](https://docs.sendsay.ru/statistics/all-about-campaign-reports) — эти показатели помогут вносить изменения и улучшать рассылки.

## Где смотреть статистику по&nbsp;сайту и&nbsp;выпускам

Статистика по&nbsp;сайту и&nbsp;выпускам триггерных сценариев выводятся во&nbsp;вкладке **Сводка** на&nbsp;странице сайта:

![Summary](/img/ecom/ecom-statistics/summary.png)

В&nbsp;статистике по&nbsp;сайту выводятся данные:

- Уникальные контакты&nbsp;&mdash; это число подписчиков, которые посещают сайт и&nbsp;чьи данные мы&nbsp;знаем. Например, когда подписчик переходит на&nbsp;сайт из&nbsp;письма, мы&nbsp;можем отследить историю событий и&nbsp;его поведение на&nbsp;сайте.

  [Как посмотреть историю событий сайта](https://docs.sendsay.ru/ecom/how-to-connect-ecom#как-посмотреть-историю-событий-сайта)

- Анонимные контакты&nbsp;&mdash; те&nbsp;пользователи, которые &laquo;гуляют&raquo; по&nbsp;сайту и&nbsp;данных о&nbsp;которых у&nbsp;нас нет. Например, они не&nbsp;подписаны на&nbsp;рассылку или могли перейти на&nbsp;сайт не&nbsp;из&nbsp;письма.

  Если настроена передача событий **Регистрация на&nbsp;сайте** и&nbsp;**Авторизация на&nbsp;сайте**, в&nbsp;Sendsay будут передаваться email-адреса пользователей, которые зарегистрировались или авторизовались на&nbsp;сайте. Система сохранит такой адрес и&nbsp;свяжет с&nbsp;ним все поступающие события этого пользователя&nbsp;&mdash; таким образом анонимные контакты перестанут быть анонимными и&nbsp;станут участниками сценария.

:::tip Важно
Система хранит данные анонимных пользователей. Когда анонимный контакт перейдёт из&nbsp;письма на&nbsp;сайт, данные соединятся, и&nbsp;станет доступна история событий для подписчика.
:::

- Сделано заказов&nbsp;&mdash; количество совершенных заказов.
- Сумма&nbsp;&mdash; общая сумма покупок всех совершенных заказов.

Ниже&nbsp;&mdash; общая сводка по&nbsp;рассылкам, где выводится статистика по&nbsp;отправке, доставке, открытиям и&nbsp;кликам всех выпусков сценария. Статистика по&nbsp;каждому отправленному выпуску выводится в&nbsp;таблице&nbsp;&mdash; её&nbsp;можно экспортировать в&nbsp;формате `.xlsx` или `.csv`:

![Mailing list summary](/img/ecom/ecom-statistics/mailing-list-summary.png)

Здесь можно настроить фильтр по&nbsp;периоду и&nbsp;увидеть статистику по&nbsp;выпускам за&nbsp;сегодня, вчера, последнюю неделю, месяц или за&nbsp;всё время:

![Period filter](/img/ecom/ecom-statistics/period-filter.png)

Также можно отфильтровать таблицу по&nbsp;выпускам&nbsp;&mdash; чтобы посмотреть статистику конкретного письма:

![Mailing list filter](/img/ecom/ecom-statistics/mailing-list-filter.gif)

## Статистика по&nbsp;заказам и&nbsp;выручке

Если вы&nbsp;используете [полную версию модуля &laquo;Продажи](https://docs.sendsay.ru/ecom/how-to-connect-ecom#возможности-версий)&raquo;, в&nbsp;таблице по&nbsp;отправленным выпускам вам доступна статистика по&nbsp;заказам и&nbsp;выручке:

![Statistics on orders and revenue](/img/ecom/ecom-statistics/statistics-on-orders-revenue.png)

- Заказы&nbsp;&mdash; сколько заказов сделали подписчики, перешедшие на&nbsp;сайт из&nbsp;каждого конкретного письма.
- Выручка&nbsp;&mdash; этот показатель выводит сумму покупок в&nbsp;денежном эквиваленте, которые совершили подписчики, перешедшие на&nbsp;сайт из&nbsp;каждого конкретного письма.

Таким образом можно определить, сколько заказов и&nbsp;денег принесла каждая рассылка сценария. Статистику по&nbsp;заказам и&nbsp;выручке можно найти также во&nbsp;вкладке **Сводка** на&nbsp;странице сценария и&nbsp;в&nbsp;разделе **Статистика**.

Сводка по&nbsp;заказам и&nbsp;выручке также выводится в&nbsp;отчёте выпуска и&nbsp;в&nbsp;шаблоне&nbsp;&mdash; для пользователей бесплатной версии модуля &laquo;Продажи Cтарт&raquo; данные доступны только за&nbsp;последние 48&nbsp;часов.

:::tip Важно
Если вы&nbsp;используете бесплатную версию модуля &laquo;Продажи Старт&raquo;, статистика по&nbsp;заказам и&nbsp;выручке в&nbsp;сводной таблице рассылок будет недоступна. Чтобы в&nbsp;статистике выводились эти показатели, необходимо перейти на&nbsp;версию модуля &laquo;Продажи&raquo; за&nbsp;дополнительную плату&nbsp;&mdash; для этого напишите в&nbsp;чат техподдержки.

[Таблица сравнения версий](https://docs.sendsay.ru/ecom/how-to-connect-ecom/#возможности-версий)
:::

## Статистика по&nbsp;сценарию

Помимо статистических данных по&nbsp;сайту и&nbsp;выпускам, система выводит статистику по&nbsp;каждому сценарию&nbsp;&mdash; во&nbsp;вкладке **Сводка** на&nbsp;странице сценария:

![Workflow statistics](/img/ecom/ecom-statistics/workflow-statistics.png)

Здесь в&nbsp;блоке **Статистика по&nbsp;сценарию** выводятся данные о&nbsp;его участниках&nbsp;&mdash; сколько подписчиков проходят и&nbsp;завершили сценарий. А&nbsp;в&nbsp;блоке **Продажи** можно увидеть, сколько заказов и&nbsp;на&nbsp;какую сумму принёс сценарий (для пользователей бесплатной версии модуля &laquo;Продажи Cтарт&raquo; данные доступны только за&nbsp;последние 48&nbsp;часов).

В&nbsp;сводной таблице по&nbsp;отправленным выпускам также выводятся данные о&nbsp;заказах и&nbsp;выручке по&nbsp;каждому конкретному выпуску (для пользователей полной версии модуля &laquo;Продажи&raquo;). Эту таблицу можно отфильтровать по периоду, по&nbsp;конкретному письму, отправленному в&nbsp;сценарии, а также экспортировать в формате `.xlsx` или `.csv`:

![Mailing list filter](/img/ecom/ecom-statistics/mailing-list-filter.png)
